# [  Учебный проект "Табло теннисного матча"  ](https://zhukovsd.github.io/java-backend-learning-course/projects/tennis-scoreboard/)

## Основные функции

-  ✅ Создание матча
-  ✅ Добавление игроков
-  ✅ Подсчёт очков, выод на табло

## Подсчёт очков

- Матч играется до двух сетов
- При счёте 6/6 в сете, играется тай-брейк до 7 очков

##  Установка

- Клонируйте репозиторий https://github.com/SamoylovSt/tennis-match-scoreboard.git
- Установите томкат 11
- Запустите приложение

## Используемые технологии

- Java 21
- H2 in memory database
- Hibernate
- Java EE
- JUnit 5
